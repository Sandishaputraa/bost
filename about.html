How to Use

1

Enable USB Debugging

Go to Settings → About Phone → Tap Build Number 7 times → Developer Options → Enable USB Debugging

2

Connect Device

Connect Android device to PC via USB cable. Allow USB debugging when prompted.

3

Generate Commands

Use this app to generate ADB commands based on your needs.

4

Execute Commands

Copy the commands and execute in terminal/command prompt with ADB installed.
