What is DPI?
DPI = Dots Per Inch
Measures pixel density on screen

DPI Effect:
Higher DPI = Smaller UI
Lower DPI = Larger UI

Default Values:
Most phones: 360-440 DPI
Tablets: 240-320 DPI
