ADB Helper

ADB Commands Helper

Kumpulan command ADB yang berguna

Basic Commands

Check Connected Devices

adb devices
Cek device yang terhubung via USB/WiFi

Enter ADB Shell

adb shell
Masuk ke Android shell untuk eksekusi command langsung

Reboot Device

adb reboot
Restart device Android

Display & Resolution

Check Current Resolution

adb shell wm size
Cek resolusi layar saat ini

Check Current DPI

adb shell wm density
Cek DPI density saat ini

Reset Resolution & DPI

adb shell wm size reset adb shell wm density reset
Reset ke pengaturan default pabrik

Screenshot & Recording

Take Screenshot

adb shell screencap -p /sdcard/screenshot.png
Ambil screenshot dan simpan di /sdcard/screenshot.png

Record Screen

adb shell screenrecord /sdcard/record.mp4
Mulai merekam layar (tekan Ctrl+C untuk berhenti)

File Operations

Pull File from Device

adb pull /sdcard/file.txt .
Copy file dari device ke PC (folder saat ini)

Push File to Device

adb push file.txt /sdcard/
Copy file dari PC ke device (ke folder /sdcard/)

ADB Information

Requirements:
• PC/Laptop dengan ADB
• USB Cable
• USB Debugging aktif

Warning:
• Gunakan dengan hati-hati
• Backup data penting
• Jangan gunakan command aneh
